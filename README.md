# Local Business Website – Spice Garden Restaurant

## 📌 Project Overview
This project is a professional, single-page website created for a local restaurant named **Spice Garden**.  
The purpose of this website is to provide an online presence for the business, showcase its menu, and make it easier for customers to contact and visit the restaurant.

The website was created as part of **Task 3 of the Future Internship Program** to demonstrate real-world web development skills, client communication, and business problem solving.

---

## 🛠 Technologies Used
- **HTML** – for the structure of the website  
- **CSS** – for styling, layout, and design  
- **JavaScript** – for interactive and dynamic elements  
- **GitHub Pages** – for hosting the live website  

---

## 🗂️ Website Sections

The **Spice Garden** website is a **single-page site** with multiple sections, all contained in `index.html`:

1. **Home / Welcome Section**  
   - Welcomes visitors to **Spice Garden**.  
   - Features the restaurant name and a visually appealing background.

2. **Menu Section**  
   - Displays the restaurant’s dishes (appetizers, main courses, desserts, beverages).  
   - Includes an **online order option** (interactive placeholder via form or buttons).

3. **About Section**  
   - Provides information about **Spice Garden** and its story.  
   - Visitors can **book a table** through a reservation form or button.

4. **Contact Section**  
   - Allows visitors to **send queries or feedback**.  
   - Includes a contact form and optional address / phone info.

> All interactivity and navigation between sections are handled using `script.js` and styled with `style.css`.
